# UI/UX 가이드

## 기본 원칙
- 심플하고 직관적인 레이아웃
- 모바일/데스크탑 반응형 지원
- 주요 기능(검색, 커뮤니티, 상담)은 탭으로 구분

## 색상/폰트
- 네이비, 화이트, 그레이 계열 사용
- 폰트: Unbounded, sans-serif

## 컴포넌트
- Header: 상단 고정, 로고 클릭 시 홈 이동
- SearchSection: 임대인/건물 검색, 자동완성
- CommunitySection: 게시글 목록, 댓글, 추천
- ConsultSection: 상담 폼, 문의 제출
- AuthSection/LoginModal: 인증 및 로그인

## 접근성
- 버튼/링크 충분한 크기, 명확한 라벨
- 시각적 대비, 키보드 네비게이션 지원

## 사용자 흐름
- 첫 화면: 검색 탭 → 커뮤니티/상담 탭 이동 가능
- 로그인 후 글쓰기/상담 가능
