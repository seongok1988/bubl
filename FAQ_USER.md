# 사용자 FAQ

## 1. 서비스 이용 방법
- 상가 임대인 리포트는 어떻게 확인하나요?
  → 검색 탭에서 임대인 이름 또는 건물명을 입력하면 리포트 확인 가능
- 커뮤니티 게시글 작성은 어떻게 하나요?
  → 커뮤니티 탭에서 로그인 후 글쓰기 버튼 클릭

## 2. 로그인/회원가입
- 소셜 로그인은 지원하나요?
  → 카카오, 구글 등 소셜 로그인 지원
- 비밀번호를 잊었을 때 어떻게 하나요?
  → 로그인 화면에서 비밀번호 찾기 기능 이용

## 3. 리포트/상담 관련
- 임대인 평가는 익명으로 작성되나요?
  → 네, 모든 평가는 익명으로 처리됩니다
- 상담 요청은 어떻게 하나요?
  → 상담 탭에서 문의 내용 작성 후 제출

## 4. 기타
- 개인정보는 안전하게 보호되나요?
  → Supabase 인증 및 보안 정책 적용
- 문의/불편사항은 어디로 연락하나요?
  → admin@bubl.com 또는 커뮤니티 문의 게시판 이용
