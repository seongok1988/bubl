# 분쟁 대응 매뉴얼 (Legal Incident Playbook)

## 1. 접수 단계
- 접수 경로: 이메일 / 관리자 콘솔 / 내용증명
- 사건 유형 분류: 허위 주장 / 명예훼손 / 개인정보 노출 / 협박성 게시물
- 접수 즉시 상태: temporary_hidden 전환

## 2. 사실관계 검토
- lease_id 존재 여부 확인
- reviewer_type 검증
- 신고 로그 확인
- IP/계정 패턴 분석
- 기존 유사 사례 비교

## 3. 조치 원칙
- 명백한 허위 → 삭제
- 의견 표현 → 유지
- 사실 단정 불가 → temporary_hidden
- 법적 요청(수사기관) → 로그 보존 및 별도 대응

## 4. SLA 기준
- 접수 후 24시간 내 1차 조치
- 법적 요청 12시간 이내 내부 보고
- 분쟁 장기화 시 법무 검토 프로세스 전환
